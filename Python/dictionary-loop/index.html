字典 － 迴圈的應用

Posted on 2019-12-05 In Python

以迴圈遍訪字典

因為字典能放入大量的資料，所以Python允許我們使用迴圈來遍訪字典的所有內容。
字典允許用很多種方式來儲存資料，因此迴圈遍訪的方式也有幾種不同的作法，
可以迴圈遍訪字典的所有鍵 － 值對、或只遍訪所有的鍵、或只遍訪所有的值。
